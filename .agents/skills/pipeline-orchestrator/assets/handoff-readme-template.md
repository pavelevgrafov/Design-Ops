## Как попросить правку

1. **Рестайл (цвета, шрифты, отступы):** правится только `tokens.css` /
   исходные токены — разметку не трогаем (skin property).
2. **Структура (экраны, блоки, порядок):** это не рестайл — опишите,
   что изменилось в задаче пользователя, а не «как должно выглядеть».
3. **Новое состояние/флоу:** назовите экран и состояние
   (loading / empty / error / permission_denied).
4. В любом случае: одна правка = одно сообщение, ссылка на экран
   или пин-аннотация.
